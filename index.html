就是说 A 是一个所有从 0 到 N - 1 这 N 个数字的任意排列之后组成的一个数组。
然后先给两个定义：

  1. 当任意 i 和 j 满足 0 <= i < j < N 以及 A[i] > A[j] 时，我们称其为一个全局倒置（Global Inversions）。
